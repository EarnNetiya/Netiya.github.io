[0] [1] [2] .... [n-1]

str1[idx1:idx2:|step]

โดยที่ str1 เป็นข้อมูลชนิด String
idx1 เป็นลำดับที่เริ่มต้นของตัวอักขระที่ต้องการใน str1
idx2 เป็นลำดับที่สุดท้าย + 1 ของตัวอักขระที่ต้องการใน str1
step เป็นจำนวนตำแหน่งที่ต้องการข้าม + 1 (มีหรือไม่มีก็ได้)

set เป็นชนิดข้อมูลที่ประกอบด้วยรายการข้อมูลชนิดเดียวกันเรียงต่อๆ กัน ภายใต้เครื่องหมาย วงเล็บปีกกา {...} และคั่นด้วยเครื่องหมายคอมม่า (,) สามารถเป็นค่าว่างได้ และจะไม่เก็บข้อมูลที่ซ้ำ กัน สามารถเพิ่ม เปลี่ยนแปลง หรือลบข้อมูลในเซตได้เหมือนกับข้อมูลชนิด List แต่จะไม่สามารถ เข้าถึงข้อมูลด้วยเลขตำแหน่ง เนื่องจากการจัดเก็บข้อมูลใน Set จะไม่มีลำดับ

ฟังก์ชันสำหรับข้อมูลชนิด Set

ฟังก์ชัน คำอธิบาย
set1.union(set2) ให้ค่าเป็นเชตใหม่ที่ประกอบด้วยสมาชิกของ set1 และ set2 ถ้าสมาชิกซ้ำกัน จะถูกนำมาใช้เพียงตัวเดียว
set1.intersection(set2) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ที่ซ้ำกับ set2
set1.difference(set2) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 แต่ไม่เป็นสมาชิกของเชต set2
set1.add(x) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ต่อท้ายด้วยข้อมูล x
set1.remove(x) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ที่ลบสมาชิก x ออกแล้ว
set1.clear() ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ที่ลบข้อมูลทั้งหมดออก
set1.clear1() ให้ค่าเป็น set1 ที่ลบข้อมูลทั้งหมดออก
