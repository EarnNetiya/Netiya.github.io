IF
คำสั่ง if เป็นคำสั่งที่ใช้ควบคุมให้โปรแกรมตัดสินใจทำงานหรือไม่ทำงานในชุดคำสั่งที่กำหนดโดยตรวจสอบจากนิพจน์เงื่อนไขทางตรรกศาสตร์ที่กำหนดว่า เป็นจริงหรือเท็จ ถ้านิพจน์ที่กำหนดให้เป็นจริง (true) โปรแกรมจะทำงานที่ชุดคำสั่งที่อยู่หลังคำสั่ง รf แต่ถ้านิพจน์ที่กำหนดไว้เป็นเท็จ (false) โปรแกรมจะข้ามไปทำงานที่คำสั่งต่อไปทันที ซึ่งมีรูปแบบการทำงาน ดังนี้

if boolean_experssion: statements

โดยที่ boolean_expression เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
statements เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไขที่กำหนดให้เป็นจริง (true)

IF....ELSE
คำสั่ง if....else เป็นคำสั่งควบคุมให้โปรแกรมเลือกทำงานในทางเลือกใดทางเลือกหนึ่งจาก 2 ทางเลือก โดยตรวจสอบนิพจน์เงื่อนไขทางตรรกศาสตร์ที่กำหนดว่า เป็นจริงหรือเท็จ ถ้านิพจน์ ที่กำหนดให้เป็นจริง (true) โปรแกรมจะทำงานที่ชุดคำสั่งในทางเลือกหลังคำสั่ง if แต่ถ้านิพจน์ที่ กำหนดไว้เป็นเท็จ (else) โปรแกรมจะทำงานที่ชุดคำสั่งในทางเลือกหลังคำสั่ง else ซึ่งมีรูปแบบ การทำงานดังนี้

if boolean_experssion: statements else: statements

โดยที่ boolean_expression เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
statements_1 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไขที่กำหนดให้เป็นจริง (true)
statements_2 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไขที่กำหนดให้เป็นจริง (false)

IF...ELIF...ELSE
เป็นคำสั่งควบคุมให้โปรแกรมเลือกทำงานในทางเลือกใดทางเลือกหนึ่ง จากหลายทางเลือก และแต่ละทางเลือกจะมีการกำหนดนิพจน์เงื่อนไขทางตรรกศาสตร์ของแต่ละ ทางเลือกไว้ด้วย โดยโปรแกรมจะตรวจสอบนิพจน์ของแต่ละทางเลือก หากพบว่าทางเลือกไหนมี นิพจน์เป็นจริง (true) ก็จะทำงานที่ชุดคำสั่งภายในทางเลือกนั้น โดยไม่พิจารณาทางเลือกอื่นที่เหลือ อีก ซึ่งมีรูปแบบการทำงานดังนี้

if boolean_expression 1: statement_1 elif boolean expression 2: statement_2 . . . else: statement_n

โดยที่ boolean_expression เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
โดยที่ boolean_expression_ 1 เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
boolean_expression _2 เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ
เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น statement 1 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไข
boolean_expression_1 ที่กำหนดเป็นจริง (true) statement 2 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไข
boolean_ expression _2 ที่กำหนดเป็นจริง (true) statement n เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไขที่กำหนดให้ทั้งหมด เป็นเท็จ (false)

WHILE
คำสั่ง while เป็นคำสั่งควบคุมการทำงานของโปรแกรมแบบทำซ้ำที่มีการตรวจสอบเงื่อนไข ที่เป็นนิพจน์ทางตรรกศาสตร์ก่อนจะทำงานที่ชุดดำสั่งภายในลูป โดยไปรแกรมจะทำงานที่ชุดคำสั่ง ภายในลูปก็ต่อเมื่อนิพจนีที่กำหนดให้เป็นจริง และเมื่อโปรแกรมทำงานที่ชุดคำสั่งภายในลูปเสร็จสิ้น โปรแกรมจะตรวจสอบนิพจน์ใหม่อีกครั้ง และจะหยุดการทำซ้ำเมื่อนิพจนัที่กำหนดให้เป็นเท็จ ซึ่งมีรูปแบบ การทำงานดังนี้

boolean_expression: statement

โดยที่ boolean_expression เป็นนิพจน์ที่เป็นเงื่อนไขทางตรรกศาสตร์ไว้ตรวจสอบ เงื่อนไขว่าเป็นจริงหรือเท็จก่อนการทำงานเสมอ
statements เป็นชุดคำสั่งที่ต้องการให้มีการทำซ้ำ จากลักษณะการทำงานของคำสั่ง

FOR
คำสั่ง for เป็นคำสั่งควบคุมการทำงานของโปรแกรมแบบทำช้ำ ที่มีการตรวจสอบเงื่อนไขที่ เปินนิพจน์ทางตรรกศาสตร์ ก่อนการทำงานที่ชุดคำสั่งกายในลูปทุกครั้ง ซึ่งเป็นการตรวจสอบกับ ช่วงข้อมูลที่ใด้กำหนดไว้ โดยโปรแกรมจะทำงานที่ชุลดำสั่งกายในลูปเมือเงื่อนไขเป็นจริง ซึ่งจะ เพิ่มหรือลดค่าตัวแปรควบคุมตามคำสั่งที่กำหนดไว้

for count in range(value): statements

โดยที่ count ตัวแปรที่ใช้ในการวนลูป
value ช่วงข้อมูลที่เป็นตัวเลขสำหรับการทำซ้ำที่ต้องการประกอบด้วย
ข้อมูลตัวเลข 2 ค่าคือค่าเริ่มต้นและดำสุดท้ายของการทำซ้ำ+1
