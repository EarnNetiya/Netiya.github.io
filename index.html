ชนิดของข้อมูล (Data Type)

ข้อมูลชนิดตัวเลข (Number)

1.ข้อมูลเลขจำนวนเต็ม (Integer Number) ประกอบด้วยเลขจำนวนเต็มที่เป็นทั้งค่าบวก ค่าลบ และยังสามารถใช้กับเลขฐานได้ดังนี้
ㆍ เลขฐานสอง ใช้ตัวอักษร 0b นำหน้า
ㆍเลขฐานแปด ใช้ตัวอักษร 0o นำหน้า
ㆍ เลขฐานสิบหก ใช้ตัวอักษร 0x นำหน้า
2. ข้อมูลเลขจำนวนทศนิยม (Floating Point Number) ประกอบด้วยเลขจำนวนที่เป็น ทศนิยม มีเครื่องหมาย . คั่นระหว่างตัวเลข และสามารถเขียนในรูปแบบของเลขยกกำลังสิบ (Exponential Form) ได้อีกด้วย
3. ข้อมูลเลขจำนวนเชิงซ้อน (Complex Number) ประกอบด้วยเลขจำนวนเชิงซ้อน ซึ่งจะ
ถูกใช้ในงานทางวิทยาศาสตร์และวิศวกรรม โดยเขียนอยู่ในรูปแบบของคู่ x*yi โดยเรียก x ว่า
จำนวนจริง และเรียก y ว่า ส่วนจินตภาพ และจะใช้ j แทน i

ตัวดำเนินการ (Operator)

ตัวดำเนินการกำหนดค่า (Assignment Operator)ตัวดำเนินการประเภทนี้ ใช้สำหรับกำหนดค่าให้กับตัวแปรทางด้านซ้ายของตัวดำเนินการ

ตัวดำเนินการ คำอธิบาย
= เท่ากับ : การนำค่าตัวถูกกระทำที่ได้จากด้านขวาของตัวดำเนินการ มากำหนดค่าให้กับ ตัวถูกกระทำทางด้านซ้ายของตัวดำเนินการ
+= บวกเท่ากับ:กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับผลบวก ระหว่างค่าตัวถูกกระทำ ทางด้านซ้ายกับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
-= ลบเท่ากับ : กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับผลลบ ระหว่างค่าตัวถูกกระทำ ทางด้านซ้ายกับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
*= คูณเท่ากับ : กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับผลคูณ ระหว่างค่าตัวถูกกระทำ ทางด้านขวากับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
/= หารเท่ากับ :กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับผลหาร ระหว่างค่าตัวถูกกระทำ ทางด้านซ้ายกับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
//= หารต้องการผลจำนวนเต็มเท่ากับ : กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับผลหาร ที่มีค่าเป็นจำนวนเต็ม ระหว่างค่าตัวถูกกระทำทางด้านซ้ายหารกับค่าตัวถูกกระทำทางด้าน ขวาของตัวดำเนินการ
%= หารต้องการเศษเท่ากับ :กำหนดค่ตัวถูกกระทำทางด้านซ้ายให้เท่ากับเศษที่ได้จากการหาร ระหว่างค่าตัวถูกกระทำทางด้านซ้ายหารกับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
**= ยกกำลังเท่ากับ : กำหนดค่าตัวถูกกระทำทางด้านซ้ายให้เท่ากับค่าตัวถูกกระทำทางด้านซ้าย ยกกำลังด้วยค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ
ตัวดำเนินการ คำอธิบาย
+ บวก : ทำการบวกค่าตัวถูกกระทำทางด้านช้ายกับค่าตัวถูกกระทำทางด้าน ขวาของตัวดำเนินการ
- ลบ : ทำการลบค่าตัวถูกกระทำทางด้านซ้ายกับค่าตัวถูกกระทำทางด้าน ขวาของตัวดำเนินการ
* คูณ : ทำการคูณค่าตัวถูกกระทำทางด้านซ้ายกับค่าตัวถูกกระทำทางด้าน ขวาของตัวดำเนินการ
/ หาร : ทำการหารค่าตัวถูกกระทำทางด้านซ้ายกับค่าตัวถูกกระทำทางด้าน ขวาของตัวดำเนินการ
// หารต้องการผลจำนวนเต็ม : ทำการหารค่าตัวถูกกระทำทางด้านซ้าย กับค่าตัวถูกกระทำทางด้านขวาของตัวดำเนินการ โดยให้ผลลัพธ์เป็นจำนวนเต็มที่หารได้
% หารต้องการเศษ : ทำการหารค่าตัวถูกกระทำทางด้านช้ายกับค่าตัวถูก กระทำทางด้านขวาของตัวดำเนินการ โดยให้ผลลัพธ์เป็นเศษที่เหลือของ การหาร
** ยกกำลัง : ทำการยกกำลังค่าตัวถูกกระทำทางด้านซ้าย (ฐาน) ด้วยค่าตัว ถูกกระทำทางด้านขวา (เลขชี้กำลัง) ของตัวดำเนินการ

การเปิดไฟล์ ในภาษา Python

ก่อนที่จะเริ่มทำงานกับไฟล์ สิ่งแรกที่คุณต้องทำก็คือต้องเปิดไฟล์ขึ้นมาก่อน ในภาษา Python ใช้ฟังก์ชัน open() สำหรับเปิดไฟล์เพื่อวัตถุประสงค์ที่ต้องการ โดยฟังก์ชันสามารถเปิดเพื่อทำงานได้กับทั้ง Text file และ Binary file นี่เป็นรูปแบบของการใช้งานฟังก์ชันเพื่อเปิดไฟล์

f = open(filename, mode)

โดยที่ filename เป็นชื่อหรือ Path ของไฟล์ที่ต้องการเปิด
mode เป็นโหมดสำหรับเปิดไฟล์ เช่น เปิดไฟล์เพื่อการอ่านหรือการเขียน เป็นต้น เมื่อการเปิดไฟล์สำเร็จ ฟังก์ชันจะส่งไฟล์ออบเจ็คกลับมาเพื่อนำไปใช้ในการอ่านและเขียนไฟล์ต่อไป สำหรับโหมดในการเปิดไฟล์นั้นมีหลายแบบที่คุณสามารถใช้ได้ นี่เป็นโหมดการเปิดไฟล์ที่ใช้กับฟังก์ชัน open() ในภาษา Python

Mode Description
r เปิดไฟล์เพื่ออ่านข้อมูลจากไฟล์

List และ Tuple เป็นชนิดข้อมูลที่ประกอบด้วยรายการข้อมูลชนิดใดก็ได้เรียงต่อๆ กัน โดย ข้อมูล List จะอยู่ภายในเครื่องหมายวงเล็บก้ามปู [...] ส่วนข้อมูล Tuple จะอยู่ภายในเครื่องหมาย วงเล็บ (...) และคั่นด้วยเครื่องหมายคอมม่า (,) เหมือนกัน ความแตกต่างระหว่าง List กับ Tuple คือ สามารถเพิ่ม เปลี่ยนแปลง หรือลบข้อมูลใน List ได้ แต่ไม่สามารถเพิ่ม เปลี่ยนแปลง หรือลบข้อมูลใน Tuple

การเข้าถึงข้อมูลชนิด List และ Tuple

list1(idx] หรือ tuple1[idx]

list1 เป็นข้อมูลชนิด List
tuple1 เป็นข้อมูลชนิด Tuple
idx เป็นลำดับที่ของรายการข้อมูลที่ต้องการใน list1 หรือ tuple1

2.การเข้าถึงกลุ่มข้อมูลหรือข้อมูลที่มากกว่า 1 รายการ มีรูปแบบการใช้งานดังนี้

list1(idx1:idx2:|step] หรือ tuple1[idx1:idx2: |step]

list1 เป็นข้อมูลชนิด List

Dictionary เป็นชนิดข้อมูลที่ประกอบด้วยรายการข้อมูลชนิดใดก็ได้เรียงต่อๆ กัน ภายใน เครื่องหมายวงเล็บปีกกา {...}และคั่นด้วยเครื่องหมายคอมม่า (,) โดยข้อมูลใน Dictionary ประกอบ ด้วย 2 ส่วน ดังนี้

dict1 = {key_1: value_I, key_2: value_2, ..., key_n: value_n}

dict1 เป็นข้อมูลชนิด Dictionary
key_1, key_2,..., key_n เป็นข้อมูลที่เป็นตัวชี้
value_1,value_2,...,value_n เป็นข้อมูลชนิดใดก็ได้ ที่สามารถอ้างอิงมาจาก key

statements เป็นชุดคำสั่งที่ต้องการจัดการกับข้อมูล data

statements

set1.add(x) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ต่อท้ายด้วยข้อมูล x
set1.remove(x) ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ที่ลบสมาชิก x ออกแล้ว
set1.clear() ให้ค่าเป็นเซตใหม่ที่ประกอบด้วยสมาชิกของ set1 ที่ลบข้อมูลทั้งหมดออก
