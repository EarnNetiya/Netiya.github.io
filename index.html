คำสั่ง if เป็นคำสั่งที่ใช้ควบคุมให้โปรแกรมตัดสินใจทำงานหรือไม่ทำงานในชุดคำสั่งที่กำหนดโดยตรวจสอบจากนิพจน์เงื่อนไขทางตรรกศาสตร์ที่กำหนดว่า เป็นจริงหรือเท็จ ถ้านิพจน์ที่กำหนดให้เป็นจริง (true) โปรแกรมจะทำงานที่ชุดคำสั่งที่อยู่หลังคำสั่ง if แต่ถ้านิพจน์ที่กำหนดไว้เป็นเท็จ (false) โปรแกรมจะข้ามไปทำงานที่คำสั่งต่อไปทันที ซึ่งมีรูปแบบการทำงาน ดังนี้

statements

IF...ELIF...ELSE
เป็นคำสั่งควบคุมให้โปรแกรมเลือกทำงานในทางเลือกใดทางเลือกหนึ่ง จากหลายทางเลือก และแต่ละทางเลือกจะมีการกำหนดนิพจน์เงื่อนไขทางตรรกศาสตร์ของแต่ละ ทางเลือกไว้ด้วย โดยโปรแกรมจะตรวจสอบนิพจน์ของแต่ละทางเลือก หากพบว่าทางเลือกไหนมี นิพจน์เป็นจริง (true) ก็จะทำงานที่ชุดคำสั่งภายในทางเลือกนั้น โดยไม่พิจารณาทางเลือกอื่นที่เหลือ อีก ซึ่งมีรูปแบบการทำงานดังนี้

if boolean_expression 1: statement_1 elif boolean expression 2: statement_2 . . . else: statement_n

โดยที่ boolean_expression เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
โดยที่ boolean_expression_ 1 เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น
boolean_expression _2 เป็นนิพจน์เงื่อนไขทางตรรกศาสตร์ ซึ่งมีผลการตรวจสอบ
เงื่อนไขเป็นจริง (true) หรือเท็จ (false) เท่านั้น statement 1 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไข
boolean_expression_1 ที่กำหนดเป็นจริง (true) statement 2 เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไข
boolean_ expression _2 ที่กำหนดเป็นจริง (true) statement n เป็นชุดของคำสั่งที่จะทำงานเมื่อเงื่อนไขที่กำหนดให้ทั้งหมด เป็นเท็จ (false)

WHILE
คำสั่ง while เป็นคำสั่งควบคุมการทำงานของโปรแกรมแบบทำซ้ำที่มีการตรวจสอบเงื่อนไข ที่เป็นนิพจน์ทางตรรกศาสตร์ก่อนจะทำงานที่ชุดดำสั่งภายในลูป โดยไปรแกรมจะทำงานที่ชุดคำสั่ง ภายในลูปก็ต่อเมื่อนิพจนีที่กำหนดให้เป็นจริง และเมื่อโปรแกรมทำงานที่ชุดคำสั่งภายในลูปเสร็จสิ้น โปรแกรมจะตรวจสอบนิพจน์ใหม่อีกครั้ง และจะหยุดการทำซ้ำเมื่อนิพจนัที่กำหนดให้เป็นเท็จ ซึ่งมีรูปแบบ การทำงานดังนี้

boolean_expression: statement

โดยที่ boolean_expression เป็นนิพจน์ที่เป็นเงื่อนไขทางตรรกศาสตร์ไว้ตรวจสอบ เงื่อนไขว่าเป็นจริงหรือเท็จก่อนการทำงานเสมอ

STRINGS
string เป็นชนิดข้อมูลที่ประกอบด้วยตัวอักขระหลายๆตัวมาเรียงต่อๆกันเป็นข้อความภายใน เครื่องหมาย double quote ("...") หรือเครื่องหมาย single quote ('...') ผู้อ่านจะใช้ตัวเลขจำนวนเต็ม แทนลำดับที่ของตัวอักขระใดๆ ในข้อความนั้น และเรียกตัวเลขจำนวนเต็มนี้ว่า ตัวชี้ (Index) โดย ลำดับที่ของตัวอักขระจะเริ่มจาก 0 ถึง n-1 เมื่อข้อความมีจำนวนตัวอักขระเท่ากับ n ตัว

Index

[0] [1] [2] .... [n-1]
สมาชิก 1 สมาชิก 2 สมาชิก 3 .... สมาชิก n

การเข้าถึงข้อมูลชนิด String

1.การเข้าถึงตัวอักขระ 1 ตัว มีรูปแบบการใช้งานดังนี้

str1[idx]

โดยที่ str1 เป็นข้อมูลชนิด String
idx เป็นลำดับที่ของตัวอักขระที่ต้องการใน str1

2.การเข้าถึงกลุ่มอักขระหรือตัวอักขระที่มากกว่า 1 ตัว มีรูปแบบการใช้งานดังนี้

str1[idx1:idx2:|step]

โดยที่ str1 เป็นข้อมูลชนิด String
idx1 เป็นลำดับที่เริ่มต้นของตัวอักขระที่ต้องการใน str1
idx2 เป็นลำดับที่สุดท้าย + 1 ของตัวอักขระที่ต้องการใน str1
step เป็นจำนวนตำแหน่งที่ต้องการข้าม + 1 (มีหรือไม่มีก็ได้)

Example

กำหนดให้ str1 ="python programming"

Index

ตัวอย่าง คำอธิบาย ผลลัพธ์

str1[0:6] ตัวอักขระลำดับที่ 0 ถึง 5 คือตัวที่ 1 ถึง 6 python
str1[0:6:2] ตัวอักขระลำดับที่ 0 ถึง 5 และข้าม 1 ตำแหน่งคือ ตัวที่ 1 ถึง 6 และข้ามไปทุก 1 ตำแหน่ง pto
str1[7:18] ตัวอักขระลำดับที่ 7 ถึง 17 คือตัวที่ 8 ถึง 18 programming
str1[7:18:2] ตัวอักขระลำดับที่ 7 ถึง 17 และข้าม 1 ตำแหน่งคือ ตัวที่ 8 ถึง 18 และข้ามไปทุก 1 ตำแหน่ง pormig

str1.ljust(n |,"x") ให้ค่า str1 ที่จัดรูบแบบข้อความชิดข้ายเท่ากับจำนวน n ตำแหน่ง และเติมด้วย อักขระ 'x" ทางด้านขวา ในกรณีที่จำนรนหลักของข้อความยังไม่ครบ n หลัก
